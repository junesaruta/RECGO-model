# RECGO
โครงงานนี้ต้องการศึกษาและวิเคราะห์พฤติกรรมการเลือกซื้อของสมาชิกร้านสหกรณ์จุฬาลงกรณ์
มหาวิทยาลัยเพื่อพัฒนาระบบแนะนำสินค้าให้กับสมาชิกรายบุคคลและแสดงผลลัพธ์ของการแนะนำสินค้าผ่าน
เว็บแอปพลิเคชันระบบแนะนำสินค้าโดยผู้พัฒนานำข้อมูลการซื้อสินค้าของสมาชิกร้านสหกรณ์จุฬาลงกรณ์มหาวิทยาลัย จำนวนกว่า 1,000 คน ตั้งแต่ วันที่ 1 มกราคม พ.ศ. 2567 ถึง วันที่ 31 สิงหาคม
พ.ศ. 2568 มาใช้ในการเรียนรู้ลำดับพฤติกรรมการซื้อและสร้างแบบจำลองของระบบแนะนำด้วยวิธี BERT4Rec เพื่อทำนายแนวโน้มความสนใจสินค้าของสมาชิกแต่ละคน
